🎵 How to Play

Note Guru — White Keys Edition

🎓 Your Note Reading Journey Starts Here
Note Guru trains you to read music notes and play those notes on the piano or keyboard — two core skills every musician needs.

Music is written on five lines called the stave or staff. Notes sit on or between those lines, and each note has a letter name.

The game starts with the first five notes, beginning from middle C. Watch how each note on the stave matches up with a key on the piano.
🔊

Tap to start

🎹 Notes on the Staff — All 7 White Keys
The seven white key notes — C D E F G A B — repeat in pattern all the way up the piano.

Each note has a fixed position on the staff. Watch them cycle through, one at a time — staff position, piano key, and name all lighting up together.
🔊

Tap to start

🔤 Note Names
From the home screen, select Note Names — then choose which mode you'd like to enter. You have three options.
After selecting Note Names, pick your mode:
⏱ Timed Mode
Play the game for a set amount of time — 30s, 60s, 90s, or Unlimited for an ongoing note sequence. Answer as many notes as you can before the clock runs out.
🎛️ Custom Mode
Choose exactly which notes appear on the stave. If you keep getting F and G mixed up — select just those two and drill them until they stick.
❤️ Survival Mode
Race against the timer bar — tap your answer before it runs out. Every 10 correct answers moves you up a belt and the game speeds up. Work through all 7 belts to earn the badge.
▶

Tap to start

🎹 Piano Mode
Choose Piano instead of Note Names and all three modes — Timed, Custom, and Survival — work exactly the same way. The only difference is you answer by tapping the correct key on the piano keyboard rather than a letter button.
▶

Tap to start
